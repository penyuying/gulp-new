replaceItem(item, tempPkg, reg)

替换字符串里面的变量

Parameters:
Name Type Description
item string

要被替换的字符串
tempPkg String

从对象中查找内容
reg RegExp

正则表达式
Returns:

返回替换好后的字符串

Type
string
Examples
var obj={
   a:"abcString",
   b:"{#a#}  这里第一个"
}
 
replaceItem(obj["b"], obj, /\{\#([^}]+)\#\}/ig);

//返回:
{
   a:"abcString",
   b:"abcString  这里第一个"
}

 
var obj1={
   a:{
       c:"adb",
       d:"abcString",
   },
   b:"{#a.c#}  这里第二个"
}

replaceItem(obj1["b"], obj1, /\{\#([^}]+)\#\}/ig);

//返回:
{
   a:{
       c:"abc",
       d:"abcString",
   },
   b:"abc 这里第二个"
}
